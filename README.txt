Overview of the installation and operation:

1) Establish the python environment by either:
  a) source the anaconda environment file 
    $$ conda env create -f environment.yml
 or
  b) pip install the following packages:

pip install oauth2client.service_account
pip install gspread
pip install argparse
pip install pydrive

2) Ensure that your computer local time is updated (fixing this soon to be internet time)
3) Open file "RunSetup.sh"
4) Follow instructions ins RunSetup.sh
